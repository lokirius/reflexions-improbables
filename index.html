Benvingut/da a GitHub Pages.

Aquest generador automàtic de pàgines és la forma més fàcil de crear pàgines agradables per tots els teus projectes. Redacta aquí el contingut de la teva pàgina fent servir una versió de Markdown amb l'estil de Github, selecciona una plantilla preparada per un dissenyador i publica. Després que la teva pàgina ha estat generada, pots verificar la nova branca: 

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

Designer Templates

Hem preparat algunes plantilles molt atractives pel teu ús. Segueix endavant fins les capes d'estil per navegar a través d'elles. Pots tornar enrera fàcilemnt per editar la teva pàgina abans de publicar-la. Després de publicar la teva pàgina, pots revisitar el generador de pàgines i canviar a un altre tema. El contingut de la teva pàgina es conservarà sempre que estigui en format markdown.

Rather Drive Stick?

Si prefereixes no utilitzar el generador automàtic, fes una branca anomenada gh-pages al teu repositori per crear la pàgina manualment. A més d'acceptar contingut normal en HTML, Gituhub Pages suporta Jekyll, un generador de webs senzill i amb aparença de blog, escrit pel nostre propi Tom Preston-Werner. Jekyll facililita el crear la capçalera i peu de pàgina del blog sense haver-los de copiar a cada pàgina. També ofereix suport intel·ligent per a blogs i altres característiques avançades per a plantilles.
